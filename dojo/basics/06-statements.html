If / Else / Else If

Conditional statements are used to check variables and/or their values for executing only targeted code procedure parts. 

var myObject = {};
if(myObject){ ... } // will be executed
if(typeof myObject == 'object'){ ... } // will be executed
if(typeof myObject != 'object'){ ... } // will NOT be executed

You are also able to combine different statements to handle fallback methods and/or to handle error cases. 

var OneOrZero = Math.round(Math.random()); // a 'normal' Number, 1 or 0

if(OneOrZero == 1){
	// will be executed if the value is 1
}else if(OneOrZero == 0){
	// will be executed if the value is 0
}else{
	// will be executed if no previous condition matches
}
