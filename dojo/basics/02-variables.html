Declaring Variables

In programming languages, the process of creating a variable is called declaring a variable. You can declare variables in JavaScript using the var keyword. 

var x; // declaring a variable
var y = 5; // declaring a variable and assigning a value to it

Local Variables

A variable declared within a function becomes a local variable and can only be accessed within that function. The variable has a local scope.

You can have multiple local variables with the same variable name in different functions, because local variables are only recognized by the function in which they are declared.

Global Variables

Variables declared outside a function (scope) becomes global and all scripts and functions on the web page can access it.

A variable declared without the var keyword will become a global variable by default. 

function myFunc1(){
	var myVar = 3; // local variable
	alert(myVar); // 3

	myVar2 = 5; // global variable
}
myFunc1(); // calling first function

function myFunc2(){
	alert(myVar); // undefined

	var myVar = 4; // declare local variable
	alert(myVar); // 4

	alert(myVar2); // 5
}
myFunc2(); // calling second function
