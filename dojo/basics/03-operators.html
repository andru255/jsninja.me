Operators

Arithmetic Operators

Arithmetic operators are used to perform arithmetic between variables and/or values.
If your variable y=4, the table below explains the arithmetic operators.

Operator Description Example Result
+ Addition x = y+3 x = 7 y = 4
- Substraction x = y-3 x = 1 y = 4
* Multiplication x = y*3 x = 12 y = 4
/ Division x = y/2 x = 2 y = 4
% Modulus (division remainder) x = y%3 x = 1 y = 4
++ Increment x = ++y x = 5 y = 5
x = y++ x = 4 y = 5
-- Decrement x = --y x = 3 y = 3
x = y-- x = 4 y = 3

Assignment Operators

Assignment operators are used to assign values to JavaScript variables.
If your variable x=4 and y=3, the table below explains the assignment operators.

Operator Example is the same as Result
= x = y   x = 3
+= x += y x = x+y x = 7
-= x -= y x = x-y x = 1
*= x *= y x = x*y x = 12
/= x /= y x = x/y x = 1.3333
%= x %= y x = x%y x = 1
